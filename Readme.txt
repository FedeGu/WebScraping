La idea del proyecto es analizar las marcas y precios de los celulares en una página de venta de electrodomésticos (Para éste caso elegimos la  página de fravega)

Para la primeta parte en la recolección de los datos utilizamos de las librerias de Python Request (para solicitar los datos a la página) BeautifulSoup (para especificar la solicitud) y Pandas (para estructurar los datos en tablas)


The idea of ​​the project is to analyze the brands and prices of cell phones on a home appliance sales page (For this case we chose the fravega page)

For the first part of data collection, we use the Python Request libraries (to request the data from the page), BeautifulSoup (to specify the request) and Pandas (to structure the data in tables).
